JWT Authentication API
A simple and secure User Authentication System built using Node.js, Express, MongoDB, and JWT.
This project demonstrates user registration, login, password hashing, and protected routes using JSON Web Tokens.
> Features
. User Registration
. User Login
. Password Hashing (bcrypt)
. JWT Token Authentication
. Protected Routes
. MongoDB Atlas Integration
. Environment Variables for Security 
> Tech Stack
Backend: Node.js, Express.js
Database: MongoDB Atlas
Authentication: JWT (jsonwebtoken)
Security: bcrypt
Environment: dotenv
